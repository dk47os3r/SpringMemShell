# Spring内存马研究

## 介绍
一个基于`Tomcat`的`SpringMVC`项目

用`SpringBoot`来做应该也可以，没有尝试

## 内存马检测
1. 访问`/mapping`查看已有的`mappings`
2. 访问`/test1`注册`Controller`内存马
3. 再次访问`/mapping`进行内存马检测

## 隐形马（劫持马）
1. 访问`/api`发现这是一个普通接口
2. 访问`/test2`注册隐藏内存马
3. 再次访问`/api`一切正常
4. 如果加上参数`/api?cmd=whoami`发现这是内存马
